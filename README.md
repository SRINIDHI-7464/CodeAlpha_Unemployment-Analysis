1. Introduction

Unemployment is one of the most important economic indicators used to measure the health of a country’s economy. It represents the percentage of people in the labor force who are willing to work but unable to find employment. High unemployment rates indicate economic instability and reduced growth opportunities.

In this project, Python is used to analyze unemployment data in India. The dataset was downloaded from Kaggle and contains unemployment statistics across different states and regions. The project focuses on data cleaning, analysis, and visualization to understand unemployment patterns and study the impact of Covid-19 on employment.

2. Objective

The main objectives of this project are:

* Analyze unemployment rate data in India using Python
* Perform data cleaning and preprocessing
* Identify unemployment trends across states
* Study the impact of Covid-19 on unemployment
* Generate visualizations for better insights
* Provide meaningful conclusions from the analysis

3. Dataset Description

The dataset used for this project was downloaded from Kaggle and contains unemployment statistics for different states in India.

The dataset includes the following attributes:

* Region
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

The dataset is stored in CSV format and analyzed using Python.

Dataset Name : Unemployment in India
Dataset File : `Unemployment_Rate_upto_11_2020.csv`

**Dataset Source:**
